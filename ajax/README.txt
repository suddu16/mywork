Intstructions to ensure proper working.
1. Start mysql service.
2. In mysql, run the createdatabase.sql file.
3. In the following files give your <username> and <password> inside the sql tag for the mysql database.
   a) getcomments.jsp
   b) loginValidate.jsp
   c) signup.jsp
4. start tomcat
5. Unzip the folder into webapps folder of tomcat.
6. Visit http://localhost:8080/ajax.
